Pest 是一个 PHP 测试框架，它为测试带来精美简单的语法，而不会牺牲功能。 

在 Laravel 项目中启动并运行 Pest，将在此过程中构建的真实应用程序编写测试。 

学习如何在 Laravel 项目中设置 Pest，使用 Pest 的内置断言编写测试，生成代码覆盖率等等。
